## 프로젝트 소개

>비급여 진료비 비교 플랫폼 MediPrice(메디프라이스)입니다.
심평원 공공데이터를 기반으로 주변 병원의 비급여 진료비를 비교하고, 카카오맵을 통해 위치 기반 검색을 제공합니다.
