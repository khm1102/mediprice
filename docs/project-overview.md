# MediPrice 프로젝트 기획 개요

## 서비스 소개

**MediPrice(메디프라이스)** — 비급여 진료비 비교 플랫폼

병원마다 제각각인 비급여 진료비(도수치료, 치아 임플란트, 미용 시술 등)를
지도 기반으로 한눈에 비교할 수 있는 서비스.

- **성격:** 학교 팀 프로젝트 (발표용 프로토타입)
- **데이터 출처:** 건강보험심사평가원(심평원) 공공 API

---

## 핵심 기능

### 1. 지도 기반 병원 검색

- 내 위치 또는 지정 위치 기준으로 주변 병원을 카카오맵에 표시
- 반경 필터로 검색 범위 조절
- 병원 마커 클릭 → 상세 정보 + 가격 확인

### 2. 비급여 진료비 비교

- 진료 항목(카테고리) 선택 → 주변 병원들의 가격을 리스트/카드로 비교
- 가격순 정렬로 최저가 병원 빠르게 탐색
- 심평원 API에서 가져온 실제 신고 가격 기반

### 3. 병원 상세 정보

- 병원명, 주소, 전화번호, 병원 종류
- 해당 병원의 비급여 항목별 가격 목록

---

## 사용자 유형

### 회원

- 이메일 + 비밀번호로 회원가입/로그인
- JWT 발급 → HttpOnly 쿠키 저장
- 검색 횟수 제한 없음

### 비회원

- 첫 접속 시 Guest JWT 자동 발급
- **검색 3회까지 허용**, 초과 시 로그인 유도 팝업
- 검색 횟수는 서버 메모리(ConcurrentHashMap)에서 guestId 기준 관리
- 서버 재시작 시 초기화 (프로토타입 수준)

---

## 페이지 구성

| 페이지 | URL | 설명 |
|--------|-----|------|
| 메인 | `/` | 검색 입력 + 지도 미리보기 |
| 병원 목록 | `/hospitals` | 지도 + 병원 리스트 (검색 결과) |
| 병원 상세 | `/hospitals/{id}` | 병원 정보 + 가격 목록 |
| 로그인 | `/auth/login` | 이메일/비밀번호 입력 |
| 회원가입 | `/auth/register` | 회원가입 폼 |

## 데이터 흐름

```
사용자 검색 요청 (위치 + 진료항목)
    ↓
서버: 캐시 확인 (ConcurrentMapCache)
    ├── 캐시 HIT → 바로 반환
    └── 캐시 MISS ↓
        심평원 API 호출 → 결과 캐싱 → 반환
    ↓
PostGIS로 위치 기반 필터링/정렬
    ↓
프론트: 카카오맵 마커 + 가격 리스트 렌더링
```

---

## 기획 시 참고 사항

- 프로토타입 수준이므로 결제, 리뷰, 즐겨찾기 등 부가 기능은 범위 밖
- 심평원 API 응답이 느릴 수 있으므로 캐싱이 UX에 중요
- 비회원 검색 제한은 가입 유도 목적이지 보안 목적이 아님
